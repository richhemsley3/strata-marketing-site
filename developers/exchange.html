HOW IT WORKS

Three steps to connected data.

1

Ingest

Connect external systems via REST, webhooks, or streaming.

cURL 
curl -X POST \
  https://api.strata.io/v1/webhooks \
  -H "Authorization: Bearer sk_..." \
  -d '{
    "url": "https://app.co/ingest",
    "events": ["record.*"]
  }'
2

Transform

Strata normalizes, classifies, and applies policies automatically.

Normalize
Schema mapping
Classify
PII detection
Apply Policies
GDPR, CCPA
3

Export

Push clean, governed data to any destination in any format.

cURL 
curl -X POST \
  https://api.strata.io/v1/export \
  -H "Authorization: Bearer sk_..." \
  -d '{
    "source": "src_pg_01",
    "format": "parquet",
    "dest": "s3://bucket/out"
  }'
